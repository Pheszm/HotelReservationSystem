# Hotel Reservation System

A static front-end hotel reservation system built with HTML, CSS, and JavaScript. Designed for browsing rooms, viewing services, and exploring hotel features.

> **No backend** — all forms and interactions are UI demonstrations only.

## Pages

| File | Description |
|------|-------------|
| `index.html` | Main homepage (rooms, services, contact) |
| `Html folder/login-signup.html` | Login & sign-up UI |
| `Html folder/Dashboard.html` | Guest dashboard |
| `Html folder/payment-form.html` | Payment form UI |
| `Html folder/PaymentForm.html` | Alternate payment form UI |

## Tech Stack

- **HTML5 / CSS3 / JavaScript**
- **Bootstrap** — responsive layout
- **Font Awesome** — icons
- **jQuery + SB Admin 2** — dashboard
- **Chart.js** — spending chart on dashboard

## Local Setup

Open `index.html` directly in a browser, or use XAMPP / any local server pointed at the project folder.
